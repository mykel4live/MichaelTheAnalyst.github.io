Data Cleaning in SQL

In this project, I cleaned World Layoff data as of 2023 in SQL Server. The data set described the impact of COVID 19 on countries and companies around the world as of 2023.

Steps taken to clean up dataset are:
  • Removed duplicates: using the ROW_NUMBER, Common Table Expression (CTE) i.e. WITH and DELETE statements.
  • Standardize the data; removing space at the beginning of columns using DISTNCT, LIKE, TRIM, TRAILING, UPDATE, SET, STR_TO_DATE() and ALTER TABLE statements.
  • Removed null and blank values: using JOIN and CREATE TABLE to bring the same table side-by-side, them compare and populate common columns.
  • Removed irrelevant rows and columns: using the DELETE, ALTER TABLE and DROP COLUMN statements.

Exploratory Data Analysis (EDA) in SQL

Data Exploration of World Layoffs - Impact of COVID 19 Dataset in SQL Server, using the aggregate functions; COUNT(), COUNTIF(), SUM(), AVG(), MIN(), MAX() and the WHERE, GROUP BY, ORDER BY and DESC statements.

Amongst many others, the following were explored using the dataset:
  • MAX total_laid_off and MAX percentage_laid_off
  • total_laid_off DESC
  • funds_raised_millions DESC
  • GROUP company by SUM of total_laid_off, GROUP by SUM in DESC
  • GROUP industry by SUM of total_laid_off, GROUP by SUM in DESC
  • GROUP country by SUM of total_laid_off, GROUP by SUM in DESC
  • SUBSTRING to roll up year-month in the (`date`) column
  • Using CTE ‘WITH’ to perform a Rolling_Total on year-month in the (`date`) SUM column
  • DENSE_RANK to RANK and ORDER company by their SUM(total_laid_off)

BMI CALCULATOR
PYTHON PROJECT

I was able to create a BMI calculator to collect user details and then make an informed advise based on their calculated BMI. The following data would be prompted and collected from the user: Name, Weight, Height. Therefore, BMI = (weight * 703) / (height * height).

Using if, elif and else statements, this calculation will print the user’s name and BMI in the context of their respective BMI range. Where:

  • Under 18.5 = Underweight
  • 18.5 – 24.9 = Normal Weight
  • 25 – 29.9 = Overweight
  • 30 – 34.9 = Obese
  • 35 – 39.9 = Severely Obese
  • 40 and over = Morbidly Obese
